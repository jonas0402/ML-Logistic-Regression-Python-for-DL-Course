# Logistic Regression Classification

Classification project applying logistic regression to an advertising dataset to predict user click behavior.

## Overview

Udemy course project using Scikit-learn's logistic regression model on an advertising dataset. Demonstrates logistic regression as a classification method with model evaluation using classification reports and confusion matrices.

## Approach

1. Data exploration and visualization
2. Feature engineering
3. Logistic regression model training
4. Model evaluation with classification metrics
5. Confusion matrix analysis

## Tech Stack

- **Language:** Python (Jupyter Notebook)
- **Libraries:** Scikit-learn (LogisticRegression, classification_report), Pandas, NumPy, Matplotlib, Seaborn

## Getting Started

```bash
pip install scikit-learn pandas numpy matplotlib seaborn jupyter
jupyter notebook
```
